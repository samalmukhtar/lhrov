# الكنز العربي - علوم الحروف العربية

هذا المشروع يهدف إلى استكشاف الحروف العربية وعلومها. يحتوي على واجهة تفاعلية تعرض معلومات عن كل حرف، بما في ذلك العلوم المرتبطة به وخصائصه.

## كيفية الاستخدام

يمكنك فتح ملف `index.html` في متصفحك لرؤية المشروع يعمل.

## المساهمات

إذا كنت ترغب في المساهمة، يرجى فتح طلب سحب (Pull Request) أو فتح مشكلة (Issue) لمناقشة التغييرات.
